# Git & GitHub Workshop Assignment
## What is Git?
Git is a free, open-source distributed version control system that helps 
developers track changes in their code over time. It allows you to save 
snapshots of your project called commits.

## What is GitHub?
GitHub is a cloud-based platform that hosts Git repositories online. 
It enables collaboration, code sharing, and project management for 
developers worldwide.
